5/29 Slow Productivity: Flags

Thinking through some ideas for how to attach flags to the Commands. It could be interesting to do some templates like the item below for a set of simple types like bools, ints, floats, strings (at least to start). The usage pattern could be something like the fields of a struct passed by reference similar to what you see in awesome tools like Cobra.

 1//Sample
 2
 3struct InitConfig {
 4  string directory;
 5  string branch;
 6}
 7auto cfg = InitConfig{};
 8cfg.directory = ".";  //For the current working directory
 9cfg.branch = "";  //Logic for init will set to main if this is empty
10
11auto cmd = std::make_unique<Command>("foo");
12cmd->AddFlag<string>(&cfg.directory, "d", "directory", "the directory to create the");
13cmd->AddFlag<string>(&cfg.branch, "b", "branch-name", "initial branch for the repo");
